# OrbitLLM Paper Runbook

This folder contains the current workshop-paper artifact.

## Reproduce Results

```powershell
python -m orbitllm.cli simulate
python -m orbitllm.cli plot
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
```

## Main Outputs

- `main.pdf`: 6-page English workshop paper.
- `results/simulation_metrics.csv`: main policy metrics over 10 seeds.
- `results/value_timeseries.csv`: cumulative-value traces.
- `results/ablation_metrics.csv`: ablation results.
- `results/sensitivity_metrics.csv`: energy-budget sensitivity.
- `results/generated_tables.tex`: LaTeX rows generated from CSV results.
- `figures/*.png`: all paper figures generated by `orbitllm.plot`.

## Current Main Result

Heuristic-TVD reaches about `0.90` normalized value, versus `0.59` for
All-Downlink, `0.75` for All-On-Board, and `0.85` for Fixed-Threshold.
Heuristic-TVD has zero out-of-memory and zero energy-violation events in the
main runs.

## Benchmark Note

The remote RTX 4090 24GB environment completed model downloads and profiling for
`Qwen/Qwen3-1.7B`, `Qwen/Qwen2.5-3B-Instruct`, and `Qwen/Qwen3-8B`.
Raw measured CSVs are preserved as:

- `results/profile_remote_fp16.csv`
- `results/profile_remote_int8.csv`
- `results/profile_remote_int4.csv`

`results/profile_anchor.csv` is the averaged profile used by simulation and
plotting. Replacing that file with another measured profile and rerunning the
commands above will update all plots, tables, and the PDF.
